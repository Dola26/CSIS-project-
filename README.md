# Resume Analyzer

A responsive web application that analyzes a user's resume against a job description and displays insights visually.

## Features

- Match score calculation (percentage)
- Detected and missing skills identification
- Suggestions for improvement
- Highlighted keywords in resume text
- Animated progress bar with color coding
- Responsive design (mobile + desktop)

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript (no libraries or frameworks)

## How to Run

1. Open `index.html` in a web browser
2. Paste your resume text in the first textarea
3. Paste the job description in the second textarea
4. Click "Analyze Resume"
